## Overview

BeastPay now supports direct wallet integration across multiple payment providers. Users can:
- **Connect their wallet once** and pay with multiple providers
- **Store wallet address** in browser localStorage (no server storage)
- **Switch providers** without re-entering wallet details
- **Keep TrustWallet** as planned upgrade path

**MIT Policy (No OTP Generation):** All authentication/OTP is handled by external providers. We only validate incoming signals, never generate OTP codes.

## Architecture

### 1. Wallet Connector (`wallet-connector.js`)

JavaScript plugin that manages wallet lifecycle:

```javascript
// Get wallet address
const wallet = WalletConnector.getWallet()
// Returns: { type, address, name, network, lastUpdated }

// Set wallet address
WalletConnector.setWallet('metamask', '0x...')

// Validate address
WalletConnector.isValidAddress('0x...', 'ethereum')

// Clear stored wallet
WalletConnector.clear()
```

**Storage:** Browser localStorage at key `beastpay_wallet_connector`  
**Default:** BNB USDT address (0x0582b74D10c853B52335542036e6CEA9B780849A)  
**Persistence:** Survives page reloads and browser sessions

### 2. Provider-Specific Checkout Pages

#### MetaMask (`/checkout-metamask.html`)
- Direct integration with MetaMask wallet extension
- Fee: 2.5%
- Settlement: 5-10 minutes
- Currencies: USD, EUR, GBP, AED → USDT, USDC, ETH, BTC
- **Auth Flow:** MetaMask handles signature verification (no OTP)

#### CoinRemitter (`/checkout-coinremitter.html`)
- Fast crypto delivery (3-10 minutes)
- Fee: 2.0%
- No KYC under $500
- Currencies: USD, EUR → BTC, ETH, USDT
- **Auth Flow:** Email confirmation + transaction verification

#### Plisio (`/checkout-plisio.html`)
- Multi-chain support
- Fee: 1.0%
- Settlement: 5-15 minutes
- Currencies: USD → BTC, ETH, XRP, USDT
- **Auth Flow:** Email + block confirmation (no OTP)

#### NOWPayments (`/checkout-nowpayments.html`)
- Instant settlement
- Fee: 1.5%
- Settlement: 1-30 minutes
- Currencies: USD → BTC, ETH, USDT
- **Auth Flow:** Email + blockchain confirmation

#### Stripe (`/checkout/stripe`)
- Credit/debit card payments
- Fee: 2.9%
- Instant settlement
- Currencies: USD, EUR, GBP → Fiat (not crypto direct)
- **Auth Flow:** 3D Secure, CVV validation (no OTP generated by us)

#### Transak (`/buy`)
- Fiat-to-crypto ramp
- Fee: 1.0%
- Settlement: 5-10 minutes
- Currencies: USD, AED → USDC, BTC, ETH
- **Auth Flow:** Transak SDK handles KYC (no OTP from us)

#### TrustWallet (Coming Soon)
- Placeholder for future integration
- Estimated: Q3 2026

## API Endpoints

### Create Payment (Provider-Specific)

**MetaMask:**
```bash
POST /api/payments/metamask/create
Content-Type: application/json

{
  "amount_fiat": 100,
  "fiat_currency": "USD",
  "crypto_currency": "USDT",
  "wallet_address": "0x0582b74D10c853B52335542036e6CEA9B780849A",
  "customer_email": "user@example.com"
}

Response:
{
  "checkout_url": "/checkout/metamask?payment_id=pay_xyz",
  "payment_id": "pay_xyz"
}
```

**CoinRemitter:**
```bash
POST /api/payments/coinremitter/create
(same payload structure)
```

**Plisio:**
```bash
POST /api/payments/plisio/create
(same payload structure)
```

**NOWPayments:**
```bash
POST /api/payments/nowpayments/create
(same payload structure)
```

## Security Considerations

### PCI Compliance
- ✅ **No CVV/PAN storage** — card data goes directly to Stripe/provider
- ✅ **No OTP generation** — external providers handle auth
- ✅ **Wallet address only** — stored in browser localStorage
- ✅ **HTTPS enforced** — all production deployments require TLS

### Privacy
- localStorage is domain-scoped (same-origin policy)
- User can clear wallet by clicking "Change Wallet"
- No tracking of wallet address on server (stateless)

### OAuth/AuthN
- **MIT Policy:** We never generate OTP codes
- Validation flows only: Email verification codes come from external services
- Signature verification (MetaMask, etc.) handled by provider SDKs
